Asynchronous image tasks

An image task request performs an asynchronous image-related operation, such as importing or exporting an image. The request creates a disposable task resource that you poll for information about the operation's status.

After you initiate an image import or export, poll the task's status by using the instructions in "GET_getTask_tasks__taskID__Image_Task_Calls" Section 4.4.2. "Get task details" of the Rackspace Cloud Images Developer Guide until the task completes. “Get details for a task” of this guide shows an example.

When the poll response has a status of success or failure, the response includes an expiration date and time. After expiration, the disposable task resource is deleted, but the result of the task, such as an imported or exported image, neither expires nor disappears.

For more information on task statuses, see “Task statuses”“Task statuses”.

Note

Tasks in the Cloud Images API conform to the uniform task interface provided by the OpenStack Images v2 API, with each task resource containing both input and result parameters. The API design enables individual providers, like Rackspace, to easily customize these two parameters.

The "POST_importImage_tasks_Image_Task_Calls" Section 4.4.3. "Task to import image" and "POST_exportImage_tasks_Image_Task_Calls" Section 4.4.4. "Task to export image" operations described in the Rackspace Cloud Images Developer Guide show the Rackspace requirements for these parameters.

High-level process for importing an image

  1. Put the image into your Cloud Files account.

  2. Submit the asynchronous import request by using the instructions in "POST_importImage_tasks_Image_Task_Calls" Section 4.4.3. "Task to import image" of the Rackspace Cloud Images Developer Guide. “Import an image by using tasks” of this guide shows an example.

  3. The Cloud Images service begins to fetch the image from your cloud storage and to create a new image for you. This activity takes some time.

  4. Poll the task status by using the instructions in "GET_getTask_tasks__taskID__Image_Task_Calls" Section 4.4.2. "Get task details" of the Rackspace Cloud Images Developer Guide until the task completes. “Get details for a task” of this guide shows an example.

High-level process for exporting an image

  1. Determine the UUID of the image you want to export.

  2. Submit the export request by using the instructions in "POST_exportImage_tasks_Image_Task_Calls" Section 4.4.4. "Task to export image" of the Rackspace Cloud Images Developer Guide. “Export an image by using tasks” of this guide shows an example.

  3. The Cloud Images service begins to process the image, to convert it to a convenient format for a future export of the image to another cloud (or to another region of the Rackspace cloud), and to transfer the exported image to your Cloud Files account. This activity takes some time.

  4. Poll the task status by using the instructions in "GET_getTask_tasks__taskID__Image_Task_Calls" Section 4.4.2. "Get task details" of the Rackspace Cloud Images Developer Guide until the task completes. “Get details for a task” of this guide shows an example.
